April Update: Repository contains necessary files to create a CNN Image Classification Model, tabular CNN for disease state classification from tabular data, Random forest model for the same data, and initial heterogenous (NOT WORKING) graph transformation of pre-existing tabular data. This Repository will also contain the necessary files and code to transform this dataset into a Graph dataset and eventually a GNN for classification of Alzheimer's/non-Alzheimer's.
